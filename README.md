# sistema-de-gestion-de-bibliotecas
https://github.com/SantiSrz/sistema-de-gestion-de-bibliotecas.git

Crear un sistema de gestión de bibliotecas
En este proyecto, desarrollarás un sistema para gestionar los libros de una biblioteca. Este proyecto te ayudará a trabajar con estructuras de datos complejas y a practicar conceptos de programación orientada a objetos como la herencia, la encapsulación y el polimorfismo.

Algunas funciones que puedes incluir en tu sistema de gestión de bibliotecas son:

Añadir nuevos libros al catálogo de la biblioteca
Seguimiento de libros prestados y devueltos
Generar informes de libros populares o atrasados
